Readme for TemplateForAnyGaitSetup.

This folder contains the necessary files for running the AnyGait app with a personal Projects folder
placed outside the AnyGait installation folder.

This folder can be copied to anywhere on the users harddrive where the user has write permissions.

The file "AnyGaitSetup.any" contains the setup definitions which may be changed by the user such as
markerprotocal and the location and name of the users Projects folder.
The name of this file may not be changed.

The file "AnyGait.Main.any" is the one which must be loaded in order to start the AnyGait app.
