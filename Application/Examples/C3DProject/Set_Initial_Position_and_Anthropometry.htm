Set Initial Position and Anthropometry

This task can be used to do the following two initial adjustments of the model
1.) Set the initial scaling of human model
2.) Set the initial joint angles and pelvis position
1.) Set the initial scaling of human model
The initial size of the human body can be set here.
Please note that these values (except mass) may be changed if the motion is being optimized later, in that case these values are only used as the initial guess.

2.) Set the initial joint angles and pelvis position
The initial position and orientation of pelvis can be set here, additionally the joint angles can be adjusted. More detailed positioning can be done directly in the mannequin file mannequin file.
Execute Task Description
Executing the task will drive the position of pelvis and the joint angles into the specified positions. This task can be used to finetune the initial position to macth the recorded marker positions. This is needed for the subsequent analysis of parameter and motion optimization
