Create force plates

1.) Create force plates
Change the force plate definitions if those defined in the C3D file are different from the predefined plates in the model.
This can be done in the environment.any file.
The force plates types of the C3D file can be reviewed here: force plate type

If, for instance, Data={4,4,4}, the C3D file contains three plates, all of which are type 4.
More details on the different force plates types are available online: www.c3d.org

The model repository contains a selection of the most common force plate types. These can be found in the Body/AAUHuman/ToolBox/Mocap folder.
Currently the following types are available
1.) Force plate Type2:
File: Body/AAUHuman/ToolBox/Mocap/ForcePlateType2.any File: Body/AAUHuman/ToolBox/Mocap/ForcePlateType2AutoDetection.any
2.) Force plate Type3:
File: Body/AAUHuman/ToolBox/Mocap/ForcePlateType3.any File: Body/AAUHuman/ToolBox/Mocap/ForcePlateType3AutoDetection.any
3.) Force plate Type4:
File: Body/AAUHuman/ToolBox/Mocap/ForcePlateType4.any File: Body/AAUHuman/ToolBox/Mocap/ForcePlateType4AutoDetection.any
Execute Task Description
Executing this task will drive the markers and visualize the measured forces. This enables a fast review of the C3D input data. Please pay special attention to the force directions and to non-smoothness of the data.
